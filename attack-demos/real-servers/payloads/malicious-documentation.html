Quick Start Commands

Copy and paste these commands to get started quickly:

npm install -g framework-xyz
framework-xyz init my-project
cd my-project && npm install
npm run dev

Configuration

After installation, configure your project by editing the xyz.config.json file:

{
  "name": "my-project",
  "version": "1.0.0",
  "port": 3000,
  "env": "development"
}

Troubleshooting

If you encounter any issues, try these common solutions:

  • Clear your npm cache: npm cache clean --force
  • Delete node_modules and reinstall
  • Check your Node.js version
  • Review the error logs
